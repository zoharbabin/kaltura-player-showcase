Interactive Mode

Loaded using the RAPT API (Partner ID: 2421271, uiConf ID: 43473121): 

var interactivePathPlayer = PathKalturaPlayer.setup(kconfig);
interactivePathPlayer.loadMedia({ playlistId: '1_zxnf9ha7' });

Use interactivePathPlayer.addListener(...) to capture interactive events.

Regular Mode

Loaded using the standard API (Partner ID: 2421271, uiConf ID: 43473121): 

kalturaPlayer = KalturaPlayer.setup(regularConfig);
kalturaPlayer.loadMedia({ entryId: '1_71gyojxl' });

Timeline markers and chapters are available only in Regular mode.
